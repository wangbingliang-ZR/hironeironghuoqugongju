# PROJECT CONTEXT

## 项目名称
AI Content Sales OS (HIRO AI)

## 项目目标
帮助用户完成 内容 → 线索 → 跟进 → 转化 完整链路

## 核心定位
AI 增强真人运营系统，非全自动营销机器人

## MVP 目标
两周内上线可运行 MVP，快速获得真实用户反馈

## 技术栈
- Frontend: Next.js + React + TypeScript + Tailwind
- Backend: Supabase (Database + Auth + Storage + API)
- AI: DeepSeek Chat API

## 禁止方向
- 自动群发、群控
- 黑产营销、高频自动私信
- 违规爬虫、风险自动化
